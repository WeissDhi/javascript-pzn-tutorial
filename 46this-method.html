Kata Kunci This

  • Kata kunci this adalah referensi ke object milik siapa
  • Tiap lokasi kata kunci this, bisa berbeda-beda referensi pemilik nya (inner, global, object)
  • Dalam Object Method, this merupakan referensi ke object pemilik function nya
  • Di global scope dan function, this merupakan referensi ke global object (di browser biasanya Window)
