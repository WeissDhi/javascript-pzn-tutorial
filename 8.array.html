Operasi Arrays

Operasi Keterangan
array.push Menambah data ke array (create)
array.length untuk mendapatkan panjang array
array[index] mendapatkan data posis index (read)
array.[index] = value Mengubah data di posisi index (update)
delet array[index] menghapus data di posisi index, namun index tidak bergeser (delete)
